<h2>Install</h2>

### Preparations
- Clone / Download this repo. 

### Step 1
CD into repo's directory.

### Step 2
```sh
npm install
```

### Step 3
```sh
npm start
```

### Step 4
Open "http://localhost:3000/" in your browser.
